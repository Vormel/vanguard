Learning the tools

First week will be dedicated to getting everyone familiar with the required tools,
all of which can be found on the resources page.
Completing the tutorials listed on the resources page is strongly recommended.

Structure & content

During second week the structure of the website will be finalized.
This includes the number of subpages, purpose and content of said subpages
and navigation flow throuhg the website.

The structure and navigation will be outlined in mockup flow charts
and wireframes.

Work on the writen content of the website will also begin at this stage.

HTML Skeleton

Week three will be dedicated to transforming the website mockups into
HTML documents. Placeholder text and images will be used while actual
content is being worked on.

The navigation menu will be implemented according to the flowcharts
and adjusted to fit the layout. HTML structure will be implemented
to roughly match the wireframe mockups in preparation for the design stage.

During this stage, ensuring consistnecy in the use of HTML elements will be the most important part.
E.g. make sure to be consistent with how you utilize different HTML elements,
do not rendomly switch between h2 and h3 or swap div for section.

Work on text contents of the website will continue and will slowly start
to replace placeholder text.

CSS Design

Last week will be focused on the design aspect of the website, in other words - CSS.
During this stage we will work with the media team to come up with a website theme.

Placeholder pictures will be replaced with finished media pieces,
color sheme will be decided to match the clan theme and animations will be added.

At this stage the website can be considered finished.

Additional features

Once the website is finished a developer team will be selected to maintain
and innovate the website. Features to be added post completion include:

  • React.js rewrite
  • Discord sign-in
  • Backend server
  • Public API
  • Integration with SEAN
